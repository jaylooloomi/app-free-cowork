# free-claude-code

Run Claude Code for free — one line, no API key needed.

## Windows

```powershell
irm https://raw.githubusercontent.com/jaylooloomi/free-claude-code/main/setup.ps1 | iex
```

## What it does

- Installs Python, Node.js, Ollama
- Downloads free cloud models
- Installs Claude Code
- Launches Claude Code — no API key required
